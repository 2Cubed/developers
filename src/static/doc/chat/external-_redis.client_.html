redis.client

A standard node_redis client, with promisification provided by Bluebird.
See:
  • https://github.com/mranney/node_redis

Example

redis.getAync('foo').then( ... ).catch( ... );
// is equivalent to
redis.get('foo', function (err, result) { ... });
